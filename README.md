## Descripción del proyecto
Este proyecto implementa un sistema en Python para la gestión del stock de medicamentos de la empresa F@rmaTec.

El sistema permite:
- Cargar datos de medicamentos desde un archivo CSV.
- Buscar medicamentos por identificador utilizando recursión.
- Actualizar la cantidad en stock de un medicamento.
- Eliminar medicamentos caducados según la fecha actual.
- Guardar los datos actualizados en un archivo CSV.

El proyecto aplica conceptos de **programación orientada a objetos**, **recursión**, **gestión de archivos** y **trabajo colaborativo**, tal y como se solicita en el enunciado de la actividad.

## Archivos del repositorio
- `medicamento.py`: definición de la clase Medicamento.
- `gestor_stock.py`: definición de la clase GestorStock.
- `actividad3_stock_medicamentos.csv`: archivo CSV con los datos iniciales.
- `actividad3_stock_medicamentos_actualizado.csv`: archivo CSV con los datos actualizados.
- `demo_equipo.ipynb`: notebook con la demostración del funcionamiento del programa.
